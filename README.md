This script automatically installs ALL required dependencies.

## Features

* Auto OS detection
* Installs all dependencies automatically
* Panel / Wings / Both selection
* Docker auto install
* MariaDB, Redis, Nginx
* PHP auto handling
* Auto PHP socket detection
* WSL compatible

## Supported Operating Systems

* Debian 11 / 12 / 13
* Ubuntu 20.04 / 22.04 / 24.04
* Arch Linux (development only)
* Fedora / Rocky / Alma / RHEL (development only)

## Recommended Production OS

Ubuntu 22.04 LTS
